## 📋 شاشة اختيار العمر (Age Selection)

**الموقع:** `Scenes/Setup/AgeSelection.tscn`

### الخيارات المتاحة:
1. **3-6 سنوات** 👶
   - صعوبة منخفضة (0.5x)
   - وقت غير محدود (300 ثانية)
   - مفردات بسيطة
   - أمان عالي

2. **7-12 سنة** 👧
   - صعوبة متوسطة (0.75x)
   - وقت عادي (180 ثانية)
   - مفردات متوسطة
   - تعليم متدرج

3. **13-17 سنة** 🧒
   - صعوبة عالية (1.0x)
   - وقت محدود (120 ثانية)
   - مفردات متقدمة
   - تحديات معقدة

4. **18+ سنة** 👨
   - صعوبة جداً عالية (1.2x)
   - وقت قليل جداً (90 ثانية)
   - مفردات متخصصة
   - تحديات خطيرة

### كود التنفيذ:
```gdscript
func _on_age_2_pressed() -> void:
    GameManager.set_age_group(GameManager.AgeGroup.CHILD_7_12)
    _proceed_to_difficulty()
```

## ⚔️ شاشة اختيار الصعوبة (Difficulty Selection)

**الموقع:** `Scenes/Setup/DifficultySelection.tscn`

### مستويات الصعوبة:

### 😊 **سهل (Easy)**
- معاملات الأذى: 0.5x
- سرعة الأعداء: 0.7x
- وقت التفكير: أطول
- المكافآت: أكثر
- مناسب للأطفال الصغار

### 🎮 **عادي (Normal)**
- معاملات الأذى: 1.0x
- سرعة الأعداء: 1.0x
- توازن كامل
- المكافآت: عادية
- مناسب لمعظم اللاعبين

### 💪 **صعب (Hard)**
- معاملات الأذى: 1.5x
- سرعة الأعداء: 1.3x
- وقت التفكير: أقل
- المكافآت: نادرة
- للاعبين المتقدمين

### 👹 **حلم مرعب (Nightmare)**
- معاملات الأذى: 2.0x
- سرعة الأعداء: 1.8x
- لا يوجد هامش للخطأ
- المكافآت: نادرة جداً
- للاعبين المحترفين فقط

## 🔧 كيفية التوسع

### إضافة عمر جديد:
```gdscript
# في GameManager.gd
enum AgeGroup { CHILD_3_6, CHILD_7_12, TEEN_13_17, ADULT, CUSTOM }

# في get_content_for_age():
AgeGroup.CUSTOM:
    content["difficulty_multiplier"] = 0.9
    content["time_limit_seconds"] = 150
```

### تخصيص محتوى حسب العمر:
```gdscript
var age_content = GameManager.get_content_for_age("puzzle")
var difficulty = age_content["difficulty_multiplier"]
var vocab = age_content["vocabulary_level"]  # simple, intermediate, advanced, expert
```
