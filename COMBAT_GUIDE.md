# COMBAT_GUIDE.md - شرح نظام القتال الاستراتيجي

## 🎮 نظام القتال المتقدم

### 📖 نظرة عامة
نظام قتال استراتيجي متقدم يعتمد على:
- ✅ اختيار الاستراتيجيات بالصوت
- ✅ حوار ديناميكي أثناء المعركة
- ✅ ذكاء اصطناعي متطور للأعداء
- ✅ نتائج مختلفة حسب الخطة المختارة

---

## ⚔️ استراتيجيات القتال

### 1. **الهجوم الشرس 🔥 (Aggressive Attack)**
```
الأمر الصوتي: "هجوم" أو "attack"
```
**المميزات:**
- ✅ ضرر عالي جداً (25-40)
- ❌ دفاع منخفض جداً (0.5x)
- ⚠️ قد تأخذ ضررً أكثر

**السيناريو:**
```
النتيجة: تهاجم بشراسة
الخصم يتعرض لضرر 35
لكنك تتعرض لـ 25 ضرر
قيمة العتبة: 35 > 25 ✅ أفضل للخصوم الضعفاء
```

---

### 2. **الدفاع القوي 🛡️ (Strong Defense)**
```
الأمر الصوتي: "دفاع" أو "defend"
```
**المميزات:**
- ✅ دفاع قوي جداً (0.3x الضرر)
- ❌ ضرر منخفض (8-15)
- ⚠️ بطيء لكن آمن

**السيناريو:**
```
النتيجة: تتخذ موقع دفاعي
الخصم يتعرض لـ 12 ضرر
تتعرض لـ 9 ضرر فقط (مع التقليل)
قيمة العتبة: 12 < 9 ✅ الأفضل للخصوم الأقوياء
```

---

### 3. **التوازن المثالي 🎯 (Balanced Approach)**
```
الأمر الصوتي: "توازن" أو "balance"
```
**المميزات:**
- ✅ ضرر متوسط (15-25)
- ✅ دفاع متوسط (0.65x)
- ✅ أفضل خيار عام

**السيناريو:**
```
النتيجة: توازن مثالي
الخصم يتعرض لـ 20 ضرر
تتعرض لـ 16 ضرر (مع التقليل)
قيمة العتبة: 20 > 16 ✅ الخيار الأفضل للمبتدئين
```

---

### 4. **الانسحاب التكتيكي 🏃 (Tactical Retreat)**
```
الأمر الصوتي: "انسحاب" أو "retreat"
```
**المميزات:**
- ✅ 70% نسبة نجاح للانسحاب
- ✅ تجنب بقية المعركة
- ❌ 30% فشل = ضرر إضافي (30-45)

**السيناريو:**
```
النتيجة: تحاول الانسحاب
النجاح (70%): تهرب من المعركة! ✅
الفشل (30%): تتعرض لـ 40 ضرر! 💥
```

---

### 5. **تبديل الخطة 🔄 (Switch Strategy)**
```
الأمر الصوتي: "تبديل" أو "switch"
```
**المميزات:**
- ✅ تبديل إلى استراتيجية جديدة
- ⚠️ ضرر مخفض (10-18) لأن التبديل مرتبك
- ⚠️ الخصم يستغل الالتباس (18-32 ضرر)

**السيناريو:**
```
النتيجة: تتبدل الخطة بسرعة
الخصم يستغل الالتباس!
الخصم يتعرض لـ 15 ضرر (مخفف)
تتعرض لـ 25 ضرر (مزيد)
قيمة العتبة: 15 < 25 ❌ خطة محفوفة بالمخاطر
```

## 🤖 أنواع ذكاء الخصم

### 1. **المحارب الشرس (Aggressive)** 🔴
```gdscript
AI_TYPE = AGGRESSIVE
```
- يهاجم دائماً
- لا ينسحب أبداً
- يتحول لهجوم قوي عندما يضعف
```
الاستراتيجية المثالية: الدفاع القوي + الانسحاب
```

---

### 2. **الدفاعي (Defensive)** 🔵
```gdscript
AI_TYPE = DEFENSIVE
```
- يفضل الدفاع
- يتجنب الهجوم المباشر
- يشفي نفسه عند الضعف
```
الاستراتيجية المثالية: الهجوم الشرس المستمر
```

---

### 3. **الذكي (Intelligent)** 🟢 ⭐
```gdscript
AI_TYPE = INTELLIGENT (الأصعب)
```
- يحلل الموقف
- يتكيف مع استراتيجيتك
- يستخدم أفضل الحركات
```
الاستراتيجية المثالية: التنويع والمفاجأة
```

---

### 4. **الجبان (Cowardly)** 🟡
```gdscript
AI_TYPE = COWARDLY
```
- يحاول الهروب
- ضعيف القلب
- سهل الانتصار عليه
```
الاستراتيجية المثالية: هجوم مستمر
```

## 📊 مثال عملي - معركة كاملة

### **الجولة الأولى:**
```
🎮 النظام: "قد بدأت المعركة! اختر استراتيجيتك!"

🎤 اللاعب: "هجوم!" (Attack)

🎮 النظام: "هجوم شرس! 🔥"
النتيجة:
  • أنت: 35 ضرر → الخصم (100 - 35 = 65 HP)
  • الخصم: 25 ضرر → أنت (100 - 25 = 75 HP)

⚔️ الخصم (النوع: ذكي): يقرر الدفاع
```

### **الجولة الثانية:**
```
🎮 النظام: "دورك! اختر من جديد..."

🎤 اللاعب: "دفاع!" (Defend)

🎮 النظام: "دفاع قوي! 🛡️"
النتيجة:
  • أنت: 12 ضرر → الخصم (65 - 12 = 53 HP)
  • الخصم: 7 ضرر فقط → أنت (75 - 7 = 68 HP)

⚔️ الخصم: يرى أنك تدافع، فيهاجم بقوة
```

### **الجولة الثالثة:**
```
🎮 النظام: "دورك!"

🎤 اللاعب: "توازن!" (Balanced)

🎮 النظام: "توازن مثالي! 🎯"
النتيجة:
  • أنت: 20 ضرر → الخصم (53 - 20 = 33 HP)
  • الخصم: 16 ضرر → أنت (68 - 16 = 52 HP)

⚔️ الخصم: صحته منخفضة! يحاول الهروب
```

### **الجولة الرابعة:**
```
🎮 النظام: "الخصم يبدو ضعيفاً. هذه فرصتك الأخيرة!"

🎤 اللاعب: "هجوم!" (Attack)

🎮 النظام: "هجوم نهائي! 🔥"
النتيجة:
  • أنت: 38 ضرر → الخصم (33 - 38 = -5 HP)

🏆 النظام: "لقد انتصرت! أنت البطل الحقيقي! 🏆"
```

## 🎤 أوامر صوتية للقتال

### العربية:
```
هجوم     → Attack
دفاع     → Defend
توازن    → Balance
انسحاب   → Retreat
تبديل    → Switch
```

### English:
```
attack   → Aggressive
defend   → Strong Defense
balance  → Balanced
retreat  → Tactical Retreat
switch   → Switch Strategy
```

## 💡 نصائح للاعبين

✅ **اختبر الأعداء المختلفة** - كل خصم يحتاج استراتيجية مختلفة
✅ **تنويع الحركات** - لا تكرر نفس الاستراتيجية
✅ **احترس من الضعف** - لا تنسَ أن صحتك محدودة
✅ **الانسحاب ليس عيباً** - أحياناً الحكمة أفضل من الشجاعة
✅ **اختبر الذكي** - الخصم الذكي يعلمك أفضل استراتيجية
